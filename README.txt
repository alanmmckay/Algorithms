This 'algorithms' repository is a catch-all for for various unrelated functions
put together for the sake of practice and/or class. 

The folders which are labeled <programming_language>_functions are typically
for the sake of practice, as a means to become acquainted with said programming
language. Though there are some useful utilities, such as what currently exists
within python_functions.

artificial_intelligence houses a set of assignments from my AI class. These
assignments explore breadth-first-search, genetic, and minimax search 
algorithms.

The number_maze folder is an assignment from an algorithms class which is an
implementation of a graph algorithm to find the shortest path within a number
maze.
